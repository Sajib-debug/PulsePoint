# PulsePoint - Blood Bank Management System!
"PulsePoint" is a web-based Blood Bank Management System developed using **Java, JSP, Servlet, and MySQL.  
This system helps to manage donors, blood requests, stock, and administrative operations efficiently.

>> Features:
# Admin
- Login/Logout functionality
- Manage Donors (Add, Update, Delete)
- Manage Blood Stock
- View and Complete Blood Requests
# Users
- Request Blood
- Track Request Status
# Others
- Responsive JSP pages
- Simple and clean UI 

>> Technologies Used
- Frontend: JSP, HTML, CSS, JavaScript
- Backend: Java Servlet
- Database: MySQL
- IDE: Eclipse IDE
- Version Control:Git & GitHub
